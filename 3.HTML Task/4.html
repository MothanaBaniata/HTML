Block elemants

Block elements: They consume the entire width available irrespective of their sufficiency. They always start in
a new line and have top and bottom margins. It does not contain any other elements next to it.

  • Block Elements occupy the full width irrespective of their sufficiency.
  • Block elements always start in a line.
  • Block elements doesn’t allow other elements to sit behind
  • Block elements have top and bottom margin.

Imline elemants

Inline elements occupy only enough width that is sufficient to it and allows other elements next to it which are
inline. Inline elements don’t start from a new line and don’t have top and bottom margins as block elements have.

  1. Inline elements occupy only sufficient width required.
  2. Inline elements don’t start in a new line.
  3. Inline elements allow other inline elements to sit behind.
  4. Inline elements don’t have top and bottom margin
